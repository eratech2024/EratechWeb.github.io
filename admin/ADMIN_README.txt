========================================
ERATECH ADMIN PANEL - HƯỚNG DẪN NHANH
========================================

ĐĂNG NHẬP:
---------
1. Mở file: admin-login.html
2. Nhập thông tin:
   - User: Admin
   - Pass: Eratech@5845
3. Click "Đăng nhập"

TÍNH NĂNG:
---------
✓ Quản lý Tin tức (News)
  - Thêm/Sửa/Xóa bài viết
  - Đặt bài viết nổi bật
  - Xuất/Nhập dữ liệu JSON

✓ Quản lý Dịch vụ (Services)
  - Thêm/Sửa/Xóa dịch vụ
  - Đặt dịch vụ nổi bật
  - Xuất dữ liệu JSON

✓ Trình soạn thảo WYSIWYG
  - Định dạng text (Bold, Italic, Underline)
  - Thêm tiêu đề, danh sách
  - Chèn ảnh vào nội dung
  - Xem/Sửa mã HTML

BẢO MẬT:
-------
- Phiên đăng nhập: 8 giờ
- Tự động kiểm tra phiên mỗi phút
- Nút đăng xuất ở góc trên phải

BACKUP DỮ LIỆU:
--------------
⚠️ QUAN TRỌNG: Xuất dữ liệu thường xuyên!

- Nút "Xuất Tin tức" → Tải news-data.json
- Nút "Xuất Dịch vụ" → Tải services-data.json

Lưu các file JSON này để backup!

TỐI ƯU HÓA:
----------
✓ Sử dụng đường dẫn ảnh (images/abc.jpg) 
  thay vì upload để tiết kiệm dung lượng

✓ Ảnh upload tự động nén:
  - Ảnh đại diện: 800px, 70% chất lượng
  - Ảnh nội dung: 600px, 60% chất lượng

✓ Giới hạn LocalStorage: ~5-10MB
  Hệ thống cảnh báo khi > 4MB

XEM TRANG CÔNG KHAI:
-------------------
- Icon Newspaper → Xem trang Tin tức
- Icon Cogs → Xem trang Dịch vụ

HỖ TRỢ:
------
Xem hướng dẫn chi tiết: ADMIN_GUIDE.md
